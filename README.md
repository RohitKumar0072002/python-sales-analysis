# Sales Data Cleaning and Analysis (Python)

## Project Overview
This project focuses on cleaning and analyzing a messy sales dataset using Python. The goal is to transform raw, inconsistent data into a clean dataset and generate meaningful business insights using data analysis and visualization.

The project demonstrates practical data cleaning techniques that are commonly used in real-world data analyst workflows.

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## Dataset Description
The dataset contains messy sales data with multiple data quality issues such as:

- Missing values
- Inconsistent text formatting
- Missing dates
- Null values in payment methods
- Inconsistent customer names

The goal was to clean the dataset and prepare it for analysis.

---

## Data Cleaning Steps
The following steps were performed during the cleaning process:

- Removed extra spaces from text columns
- Standardized customer name formatting
- Handled missing payment values using mode
- Generated realistic random dates for missing date values
- Converted date column to proper datetime format
- Checked and handled missing values
- Removed unnecessary index columns
- Saved the cleaned dataset

---

## Exploratory Data Analysis (EDA)

The following analyses were performed:

### Top Products by Revenue
Identified the highest revenue generating products.

### Revenue by Location
Analyzed which locations generate the most sales revenue.

### Sales Distribution
Explored how sales are distributed across different products and locations.

---

## Key Insights

**Insight 1**

Products like **Cookie, Cake, and Tea** generated the highest revenue.

**Business Impact**

These products contribute significantly to overall revenue.

**Recommended Action**

The company should focus on promoting these high-performing products through marketing campaigns or bundles.

---

**Insight 2**

Locations like **Los Angeles, Miami, and New York** generated the highest sales revenue.

**Business Impact**

These cities represent strong markets for the business.

**Recommended Action**

The company should prioritize marketing efforts and expansion strategies in these high-performing regions.
